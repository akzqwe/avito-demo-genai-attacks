Объявления

Клик «Спросить поддержку» открывает чат с этим объявлением — агент загрузит его описание через инструмент get_listing.

Подать объявление

Это поле «описание» — недоверенные данные, которые прочитает LLM-агент. Сюда можно вшить indirect prompt injection: попробуй кнопки ниже.
Каждый пресет — отдельная цепочка атаки. Цветной тег показывает, какая именно. В админке найди toggle с таким же тегом — это её защита.
